# 产品总览

## 结论

Caps Nav 是一个 macOS 原生菜单栏工具，固定使用 `Caps Lock` 作为前缀键，把主键区字母映射成编辑导航动作或自定义快捷键。

## 产品边界

- 不做窗口管理。
- 不做通用自动化平台。
- 不支持自定义前缀键。
- v1 不做 App 专属配置方案、条件规则、多步宏、云同步。

## MVP 范围

- 菜单栏常驻。
- 辅助功能权限检查与引导。
- 全局键盘事件监听。
- 基于配置方案的“触发键 -> 输出动作”映射。
- `builtin` 与 `shortcut` 两类动作。
- 多配置方案加载与切换。
- 多分类设置页。
- 应用级运行总开关与可选全局开关快捷键。
- 快捷键练习训练场。

## 当前实现状态

- 已完成菜单栏入口、设置窗口、默认配置方案初始化、设置与配置方案持久化、全局事件监听与映射动作重发。
- 已完成 `builtin` 与 `shortcut` 两类动作的可视化配置。
- 已完成多配置方案管理，包括新增、复制、重命名、删除、排序与切换。
- 已完成默认方案只读保护，避免直接改动内置基线。
- 已完成内置动作“功能效果”悬停动态预览，通过浮层演示文本编辑沙盘。
- 已完成悬浮帮助框、权限说明窗口、About 窗口和主题切换。
- 已完成应用级运行总开关、全局开关快捷键、状态栏启用/暂停入口与暂停态图标区分。
- 已完成基于当前激活配置方案的快捷键练习窗口，支持认键练习与连招挑战。
