# ProfileManager

## 模块职责

- 从已加载 profile 中选择 active profile。
- 处理 active profile 切换。
- 保证 settings 中的 `activeProfileId` 与运行时状态一致。

## 代码入口

- `CapsNav/Infrastructure/ProfileManager.swift`
- `CapsNav/App/AppBootstrap.swift`

## 输入输出

- 输入：`[Profile]`、preferred active profile id。
- 输出：运行时 active profile。

## 典型调用路径

启动时：

`AppBootstrap.start()`  
-> `ProfileLoader.loadProfiles(...)`  
-> `ProfileManager.selectActiveProfile(...)`

切换时：

`MenuBarMenuView` 或 `PreferencesRootView`  
-> `AppBootstrap.switchActiveProfile(to:)`  
-> `ProfileManager.switchProfile(...)`  
-> `SettingsStore.save(...)`

## 当前边界

- 当前阶段已支持加载与切换。
- 新建、复制、删除、保存编辑后的 profile 会在下一阶段补齐。
