# Builtin Actions

## 语义动作

- `moveLeft`
- `moveRight`
- `moveUp`
- `moveDown`
- `selectLeft`
- `selectRight`
- `selectUp`
- `selectDown`
- `moveWordLeft`
- `moveWordRight`
- `selectWordLeft`
- `selectWordRight`
- `moveToLineStart`
- `moveToLineEnd`
- `selectToLineStart`
- `selectToLineEnd`
- `deleteBackward`
- `deleteForward`
- `deleteWordBackward`
- `deleteWordForward`

## 当前默认映射

- `moveLeft` -> `Left Arrow`
- `moveRight` -> `Right Arrow`
- `moveUp` -> `Up Arrow`
- `moveDown` -> `Down Arrow`
- `selectLeft` -> `Shift + Left Arrow`
- `selectRight` -> `Shift + Right Arrow`
- `selectUp` -> `Shift + Up Arrow`
- `selectDown` -> `Shift + Down Arrow`
- `moveWordLeft` -> `Option + Left Arrow`
- `moveWordRight` -> `Option + Right Arrow`
- `selectWordLeft` -> `Shift + Option + Left Arrow`
- `selectWordRight` -> `Shift + Option + Right Arrow`
- `moveToLineStart` -> `Command + Left Arrow`
- `moveToLineEnd` -> `Command + Right Arrow`
- `selectToLineStart` -> `Shift + Command + Left Arrow`
- `selectToLineEnd` -> `Shift + Command + Right Arrow`
- `deleteBackward` -> `Delete`
- `deleteForward` -> `Forward Delete`
- `deleteWordBackward` -> `Option + Delete`
- `deleteWordForward` -> `Option + Forward Delete`

## 实现说明

`deleteWordForward` 当前采用 `Option + Forward Delete`，没有显式使用 `Fn`。这是为了先保证实现简单且可验证，后续再根据真实兼容性决定是否切换方案。

## 代码入口

- `Caps Nav/System/ActionResolver.swift`
